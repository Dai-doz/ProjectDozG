# What is ProjectDozG?

  - ProjectDozG is a project that bring higher OneUI version to Old phone and unlock High-end feature for midrange variant , still a small Project that handle by Dai_Doz  

  
# Feature✨

- Based on S23FE and A26 Firmware
- GalaxyAI support
- EROFS Powered
- HighEND animation
- Brief
- Samsung DeX
- AOD but who use on a lcd screen?
- Debloated for best experience
- Knox patch integrated
- DeX via HDMI 


# ProjectDozG Feature
- OTG Update but not now
- Unlock Higher FPS option in Game(Example:Pubg)
- Play Integrity Fix integrated
